## Challengecup/Challengecup
本人本科二年级项目，为无人机团队开发的app
1. socket连接服务器，利用json传输数据
2. 利用BLE蓝牙hc08与嵌入式无人机硬件进行通讯，得到byte数据
3. 因为ble每次只能有20字节传输，所以自己写了一个类进行整合数据
4. 主线程实现数据UI显示
5. 利用tabhost进行不同activity互相切换
6. 与网页链接代码
